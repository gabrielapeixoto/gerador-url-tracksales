# Gerador de URL Tracksales

## Visão Geral
Código para criar url com name e email de quem está respondendo para que possa ser detectado essas informações a partir da API do Tracksales. Cada pesquisa é gerada automaticamente de acordo com o name e email que o usuário insere.

### Visão Geral Técnica
- Python
- HTML
- CSS
